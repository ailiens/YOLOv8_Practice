# YOLOv8 모델 사용해보기 

## 🌟 프로젝트 개요
객체 인식 모델 중 하나인 YOLO와 Streamlit을 사용하여, 해당 모델을 사용할 수 있도록 구현

## 🎯 목표
- 최신 YOLO 모델을 Streamlit을 활용하여 웹 애플리케이션으로 구현
- 이를통해 사용자가 쉽게 YOLO 모델을 경험하고 활용할 수 있게 만들기
